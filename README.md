# URL_Shortner_App_Basic
A simple web application built with **Flask** and **SQLAlchemy** that shortens long URLs into easy-to-share links. It also stores both original and shortened URLs in a database and provides a history page to view all saved links.
## Features - Shorten long URLs - Copy shortened links with one click - View history of saved URLs - Built with Flask, SQLAlchemy, HTML, CSS, and Bootstrap ## How to Run ```bash pip install flask flask_sqlalchemy validators python URL_Shortner.py
